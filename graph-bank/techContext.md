# Memory Bank: Technical Context

## Technology Stack
- **前端框架**: Angular 20.1.0 (最新版本)
- **UI 元件庫**: Angular Material 20.1.0
- **後端服務**: Firebase 11.10.0 (完整服務套件)
- **狀態管理**: Angular 內建狀態管理 + RxJS
- **認證服務**: Firebase Auth
- **資料庫**: Firestore (NoSQL)
